⌈ 回溯 ⌋

将每个数字对应的字母，加入哈希表

```java
    Map<Character, String> map = new HashMap<>() {{
        put('2', "abc");
        put('3', "def");
        put('4', "ghi");
        put('5', "jkl");
        put('6', "mno");
        put('7', "pqrs");
        put('8', "tuv");
        put('9', "wxyz");
    }};
```

每次取电话号码的一位数字，通过哈希表得到该数字对应的字母序列，将其中一个字母插入到已有字母排列的后面，继续处理字母排列的下一位置，直到得到一个完整的字母排列为止。


然后进行回溯，得到其余的字母排列。
