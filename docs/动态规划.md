# 动态规划

动态规划（`Dynamic Programming`），简称 `DP`，常常适用于有重叠子问题和最优子结构性质的问题。

基本思想：将复杂的问题分解为相对简单的子问题，然后求解子问题，再根据子问题的解得出原问题的解。

适用情况：

-   最优子结构性质
-   无后效性
-   子问题重叠性质
